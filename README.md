This is the official repository for **Agent-Based Auditing of Mathematical Proofs in Research Papers**, accepted as an Oral presentation at the [ICML 2026 Workshop AI4Research](https://ai4research-icml-workshop.github.io/).

This project provides tools and practical examples for conducting structured, agent-based audits of mathematical proofs directly from informal scientific papers (LaTeX), eliminating the need for strict formalization required by traditional theorem provers.

## 📂 Repository Structure

* **`example_1/`**:     An example LaTeX paper containing naturally occurring mathematical errors that demonstrates the complete auditing workflow. (Author information has been anonymized.)
* **`SKILL.md`**:     The complete proof-auditing skill for Claude Code. Apply this skill directly to your own LaTeX projects to perform an automated mathematical proof audit.
